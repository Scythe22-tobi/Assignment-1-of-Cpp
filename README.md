# C++ Programming Assignment – 1

## 📌 Description
This repository contains solutions to **20 basic C++ programming problems**.  
All programs are written in **C++ (.cpp)** and executed using the **GCC compiler (g++)**.

This assignment focuses on strengthening fundamental programming concepts and logical thinking.

---

## 🛠 Concepts Used
- `cin` and `cout`
- Conditional statements (`if`, `else`)
- Arithmetic operators
- Logical operators
- Character handling
- Basic mathematical calculations

---

## 🧾 List of Programs

### Q1
Scan 3 integers from user and print their average.

### Q2
Scan 3 numbers from user and find the largest of them.

### Q3
Scan 2 numbers from user and print square and cube of both numbers.

### Q4
Scan a number and print message "Correct Input" if number lies between 15 and 20.

### Q5
Scan a number and check if it is even or odd.

### Q6
Scan a number and check if it is multiple of 6 (without directly using 6).

### Q7
Scan a character and print "APPLE" if user enters 'a' or 'A', otherwise print "NO APPLE".

### Q8
Scan a character and check whether it is capital alphabet or not.

### Q9
Scan 2 integers and a choice:
- If choice = 1 → Print addition
- Otherwise → Print subtraction

### Q10
Scan a character and check whether it is digit or not.

### Q11
Scan length and width of rectangle:
- Find area and perimeter
- If area > 100 → Print "Big Rectangle"
- Else → Print "Small Rectangle"

### Q12
Scan radius of circle:
- Find area and circumference
- If area > 50 → Print how much greater than 50
- If area < 50 → Print how much smaller than 50

### Q13
Fixed Deposit Program:
- Input: Principal (P) and Years (N)
- Interest rate = 5%
- Calculate total amount
- If total > 12000 → Suggest Mutual Fund
- Else → Suggest Gold investment

### Q14
Scan gender:
- M → Male
- F → Female
- Otherwise → Wrong Input

### Q15
Scan length and width of rectangle:
- Check if it is square or rectangle.

### Q16
Scan marks of 4 subjects:
- If average > 75 → DISTINCTION
- Else → STUDY MORE

### Q17
Print ASCII value of a character.

### Q18
Scan salary:
- If salary > 10000 → Add 5% bonus
- Else → Add 8% bonus
- Print final salary

### Q19
Scan 3 integers and find the smallest among them.

### Q20
Scan a 3-digit number and print each digit separately.
